## üí° Common Visual Issues to Look For

### Layout Problems
- [ ] Excessive white space or cramped density
- [ ] Misaligned elements
- [ ] Inconsistent padding/margins
- [ ] Responsive breakpoint issues

### Typography Issues
- [ ] Font size hierarchy unclear
- [ ] Poor contrast (readability)
- [ ] Inconsistent font families
- [ ] Text overflow or truncation

### Color & Branding
- [ ] Color scheme inconsistency
- [ ] Poor accessibility (WCAG AA/AAA)
- [ ] Clashing colors in charts
- [ ] Missing brand elements

### Data Visualization
- [ ] Chart legends unclear
- [ ] Axis labels missing or tiny
- [ ] Too many colors/patterns
- [ ] Data not prominent enough

### UX Feedback
- [ ] Button states unclear (hover, active, disabled)
- [ ] Loading states missing
- [ ] Error messages not visible
- [ ] Success confirmations absent

### Professional Polish
- [ ] Placeholder text (lorem ipsum)
- [ ] Debug info visible
- [ ] Console errors showing
- [ ] Unfinished features visible
