## Production Notes

### Screen Recording Settings
- Resolution: 1920x1080
- Frame Rate: 30fps
- Codec: H.264
- Audio: 48kHz, 256kbps

### Editing Notes
- Smooth transitions between scenes (0.5s crossfade)
- Highlight cursor movements for key interactions
- Zoom to 150% for code/text readability
- Add subtle background music (low volume, non-distracting)
- Use text overlays for key metrics
- Animated typing for Claude responses (realistic speed)

### Assets Needed
- Title card graphic (professional, modern)
- Metrics dashboard mockup
- VS Code theme: Dark+ (high contrast)
- Claude Code panel visible on right side
- Terminal with clear, readable font (Fira Code, 16pt)

### Voiceover Guidelines
- Tone: Professional but approachable
- Pace: Moderate (140-160 words per minute)
- Energy: Enthusiastic but not over-the-top
- Clear pronunciation of technical terms
- Emphasize time savings numbers
