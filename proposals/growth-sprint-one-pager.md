## Outcome
Ship full Jorge suite + analytics + compliance guardrails in 2-3 weeks.

## Scope
- Lead + Buyer + Seller orchestration
- Advanced scoring and routing logic
- Compliance telemetry (SMS opt-out, frequency, score)
- Revenue and conversion analytics dashboard

## Exclusions
- White-label multi-tenant architecture overhaul
- Custom native mobile apps

## KPIs
- 90%+ qualification routing accuracy target
- Compliance score endpoint available and monitored
- Cached read endpoints P95 under 500ms

## Acceptance Tests
- v2 route contract tests for billing/prediction/journey/compliance
- Integration tests for DB path and error path
- Demo script walkthrough against production-like data

## Timeline
- Discovery + data contract lock: 2 days
- Build: 8-12 days
- QA + deployment: 3-5 days
