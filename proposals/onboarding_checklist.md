# Lead Response Copilot - Onboarding Checklist

## Pre-Kickoff
- [ ] Contract signed and billing profile confirmed.
- [ ] Client owner, technical contact, and success contact assigned.
- [ ] Access handoff window scheduled.

## Access and Data
- [ ] CRM admin access granted.
- [ ] API credentials configured.
- [ ] Messaging provider credentials configured.
- [ ] Historical lead data export validated.

## Implementation
- [ ] Tenant scope configured (`tenant_id`/`location_id`).
- [ ] v2 pilot endpoints smoke-tested.
- [ ] Response headers and envelope contract validated.
- [ ] Compliance/SLA monitors enabled.

## Launch Readiness
- [ ] Dry run complete from lead intake to handoff.
- [ ] Dashboard and KPI report pipeline verified.
- [ ] Incident escalation contact list confirmed.

## Go-Live
- [ ] Go-live completed within 5 business days of access handoff.
- [ ] Week-1 KPI proof-pack delivered.
