## Project Overview

**EnterpriseHub** is a Python-based real estate AI platform featuring GoHighLevel CRM integration, lead intelligence, and property matching automation.

### Technology Stack
| Layer | Technology | Notes |
|-------|------------|-------|
| **Frontend** | Streamlit 1.41+ | Multi-page demo app |
| **Backend** | FastAPI + Python 3.11+ | Async API services |
| **AI/ML** | Claude API, Gemini, ChromaDB | Content generation + embeddings |
| **CRM** | GoHighLevel API | Lead management, workflows |
| **Data** | Pandas, Redis, JSON | No traditional database |
| **Deploy** | Railway, Vercel | Demo + production |

### Architecture
```
ghl_real_estate_ai/
├── streamlit_demo/        # Main Streamlit application
├── services/              # Core business logic & AI services
├── api/                   # FastAPI backend routes
├── core/                  # Shared utilities & configuration
└── tests/                 # Test suite
```

## Domain Context

**Real Estate AI Specialization**:
- **Lead Scoring**: Predictive buyer/seller qualification
- **Property Matching**: AI-powered recommendation engine
- **Conversation Intelligence**: Claude-powered lead interaction analysis

**Key Business Entities**:
- **Leads**: Contact records with engagement scoring
- **Properties**: Listings with AI-generated marketing content
- **Conversations**: Chat transcripts with sentiment analysis

## Security Considerations

**Forbidden Paths (Contains Sensitive Data)**:
```
.env                          # API keys and secrets
data/analytics/*.csv          # Real customer analytics data
*.csv (root level)           # Export files may contain PII
ghl_real_estate_ai/core/config.py  # May contain hardcoded values
```

**Safe Paths**:
```
ghl_real_estate_ai/**/*.py    # All Python source code
tests/**/*.py                 # Test files
*.md                         # Documentation
.claude/**                   # Claude Code configuration
requirements*.txt            # Dependencies
```
