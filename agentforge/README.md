# AgentForge

Lightweight multi-agent LLM orchestration framework.

## Highlights

- Async-first DAG execution engine
- Extensible tool and plugin system
- Multiple memory backends
- Optional observability integrations
- Optional MCP and LLM provider integrations

## Development

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e .[dev]
pytest -q
```
