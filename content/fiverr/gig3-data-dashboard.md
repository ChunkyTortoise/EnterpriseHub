## FAQ

### Q: What file formats do you accept?

**A:** I accept CSV, XLSX, XLS, JSON, and Google Sheets exports. Premium supports database connections (PostgreSQL, MySQL) and API integrations.

---

### Q: How much data can you handle?

**A:** Basic: Up to 10,000 rows. Standard: Up to 100,000 rows with pagination. Premium: Up to 1M+ rows with aggregation. For larger datasets, I implement sampling or server-side processing.

---

### Q: Can you clean my messy data?

**A:** Yes! Data cleaning is included in all packages. I handle:
- Missing values (imputation or removal)
- Duplicate detection
- Format standardization
- Column renaming
- Data type conversion

---

### Q: Will the dashboard work offline?

**A:** Yes. All dashboards are self-contained HTML files (Premium adds PDF). No server required—just open in any browser. Premium adds option for local Streamlit deployment.

---

### Q: Can I update the dashboard with new data?

**A:** Basic: Send me new files and I'll update (1 free update). Standard: I provide a simple updater tool. Premium: Self-updating dashboard that pulls from your data source.

---

### Q: What if I need changes after delivery?

**A:** All packages include revision rounds. After that, I offer reasonable modification rates. The dashboard code is yours to modify.

---

### Q: Do you provide ML predictions?

**A:** Premium includes basic forecasting (time series) and anomaly detection. For custom ML models (recommendations, clustering, NLP), I offer consulting packages separately.
