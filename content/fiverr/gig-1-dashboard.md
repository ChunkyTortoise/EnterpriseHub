## FAQ (Copy-Paste Ready)

**Q: What file formats do you accept?**
I accept CSV, XLSX, XLS, JSON, and Google Sheets exports. Premium supports direct database connections (PostgreSQL, MySQL) and API integrations.

**Q: How much data can you handle?**
Basic: up to 10,000 rows. Standard: up to 100,000 rows with pagination. Premium: 1M+ rows with server-side aggregation and sampling. All tiers load in under 500ms.

**Q: Can I update the dashboard with new data myself?**
Basic: send me new files and I update once free. Standard: I provide a simple drag-and-drop updater tool. Premium: the dashboard auto-refreshes from your data source on a schedule you set.

**Q: Will it work offline / without a server?**
Yes. Basic and Standard deliver self-contained HTML files -- just open in any browser. Premium adds optional Streamlit deployment for teams that want a hosted URL.

**Q: What if my data is messy?**
Data cleaning is included in every tier at no extra charge. I handle missing values, duplicates, inconsistent formatting, column renaming, and type conversion automatically.
