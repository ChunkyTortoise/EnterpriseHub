## Lead Scoring

### Reply Classification

| Score | Classification | Action |
|-------|----------------|--------|
| 5 | Hot - Requested demo | Immediate call scheduling |
| 4 | Warm - Engaged, questions | Same-day response |
| 3 | Moderate - Replied, not interested | Add to nurture |
| 2 | Low - Autoreply/OOO | Follow up in 2 weeks |
| 1 | Negative - Opt-out | Remove from list |
| 0 | Bounce | Clean from list |

## Campaign Checklist

### Pre-Campaign (Days 1-3)
- [ ] Email infrastructure configured (SPF, DKIM, DMARC)
- [ ] Email templates finalized and tested
- [ ] Demo links created and working
- [ ] All 30 targets researched
- [ ] All emails verified
- [ ] Personalization notes ready
- [ ] Tracking spreadsheet set up
- [ ] A/B test plan defined

### During Campaign (Days 4-14)
- [ ] Daily send tracking
- [ ] Open rate monitoring
- [ ] Reply response within 24 hours
- [ ] A/B test data collection
- [ ] Weekly metrics review
- [ ] Follow-up sequence execution

### Post-Campaign (Day 14+)
- [ ] Full campaign analysis
- [ ] Template optimization
- [ ] Best performers identified
- [ ] Lessons learned documented
- [ ] Nurture sequence for unconverted
- [ ] Next campaign planning
