# Upwork Job Search System

**Complete workflow** for finding, evaluating, and applying to Upwork gigs without MCP complexity.

---

## Quick Start (5 minutes)

1. **Copy the Gemini search prompt**:
   ```bash
   cat content/upwork/GEMINI_SEARCH_PROMPT.md | pbcopy
   ```

2. **Paste into Gemini** at https://gemini.google.com

3. **Review results** - Gemini will find and score 10-15 jobs

4. **Track promising jobs**:
   ```bash
   python scripts/upwork_tracker.py add <url> --title "..." --rate "..." --fit 9
   ```

5. **Generate proposals** using GEMINI_PROPOSAL_PROMPT.md

6. **Submit on Upwork** and mark as applied:
   ```bash
   python scripts/upwork_tracker.py update <id> --status applied
   ```

## Tracker Script Commands

### Track a New Job
```bash
python scripts/upwork_tracker.py add "https://upwork.com/job/abc123" \
    --title "Senior RAG Engineer" \
    --rate "$70/hr" \
    --fit 9 \
    --keywords "RAG, FastAPI, Claude"
```

### List Jobs
```bash
# All jobs
python scripts/upwork_tracker.py list

# By status
python scripts/upwork_tracker.py list starred
python scripts/upwork_tracker.py list applied
python scripts/upwork_tracker.py list interview
```

### Show Job Details
```bash
python scripts/upwork_tracker.py show 5
```

### Update Job Status
```bash
# Applied
python scripts/upwork_tracker.py update 5 --status applied

# Interview scheduled
python scripts/upwork_tracker.py update 5 --status interview \
    --notes "Call scheduled Feb 18, 2pm PST. Demo AgentForge."

# Rejected
python scripts/upwork_tracker.py update 5 --status rejected \
    --notes "Client chose someone else."
```

**Valid statuses**: `starred`, `applied`, `interview`, `offered`, `accepted`, `rejected`, `withdrawn`

## Troubleshooting

**"Gemini can't find jobs"**:
- Try more specific keywords
- Use SHORT prompt for targeted search
- Search Upwork manually and paste URLs into tracker

**"Proposals feel generic"**:
- Add more job-specific details to GEMINI_PROPOSAL_PROMPT
- Ask for 3 different hooks, pick the best
- Mention specific client pain points from job description

**"Not getting responses"**:
- Check response rate (should be >15%)
- If <15% after 10 proposals, adjust messaging
- Try lower rates for first 2-3 contracts to build reviews
- Apply faster (within 1 hour of job posting)
