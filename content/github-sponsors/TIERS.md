## FAQ

**Can I change tiers later?**  
Yes, you can upgrade or downgrade at any time through GitHub Sponsors.

**Is there an annual discount?**  
GitHub Sponsors supports annual billing at ~10% discount.

**What happens if I cancel?**  
Your badge and benefits will continue until the end of the current billing period.

**Do you offer invoicing?**  
Yes, GitHub Sponsors provides receipts/invoices through your GitHub account billing settings.
