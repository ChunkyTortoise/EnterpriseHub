## Google Snippet Optimization

### Best Practices Applied

1. **Meta Description = First 160 chars of product description**
   - Ensures Google snippet matches what buyers see on Gumroad
   - Reduces bounce rate from search results

2. **Title Format**: `[Product Name] [Tier] - [Key Benefit] ([Social Proof])`
   - Example: "DocQA Engine Starter - Production RAG Pipeline (500+ Tests)"
   - Includes keyword + value prop + credibility in 60 chars

3. **URL Slug Structure**: `[product]-[tier]-[keyword]`
   - Example: `docqa-pro-rag-optimization`
   - Hyphen-separated (Google prefers over underscores)
   - 3-5 words max for readability

4. **Category Selection**:
   - Developer Tools (11 products) - technical buyers
   - Business Tools (6 products) - non-technical buyers
   - Productivity Tools (4 products) - solo creators
