| **Data Points** | Profile views, portfolio link clicks |
| **Frequency** | Weekly (every Monday) |
| **Notes** | Track both Upwork profile and external portfolio site separately |

---

### Portfolio CTA Rate

| Aspect | Details |
|--------|---------|
| **Tool** | Google Analytics 4 (GA4) |
| **Setup** | Set up event tracking for CTA clicks (contact form, proposal request, email links) |
| **Data Points** | Page views, CTA click events |
| **Frequency** | Weekly (every Monday) |
| **Notes** | Track different CTA types separately to identify best performers |

---

### Proposal Response Rate

| Aspect | Details |
|--------|---------|
| **Tool** | Upwork Proposal Dashboard, Email Client |
| **Setup** | Maintain proposal log spreadsheet; enable email read receipts |
| **Data Points** | Proposals sent, responses received, response time |
| **Frequency** | Weekly (every Monday) |
| **Notes** | Track response quality (positive, negative, neutral) |

---

### Shortlist Rate

| Aspect | Details |
|--------|---------|
| **Tool** | Upwork Proposal Dashboard |
| **Setup** | Enable Upwork notifications for shortlist events |
| **Data Points** | Proposals sent, shortlisted proposals |
| **Frequency** | Weekly (every Monday) |
| **Notes** | Correlate with proposal quality scores |

---

### Close Rate

| **Notes** | Track reasons for lost opportunities |

### Average Contract Value
