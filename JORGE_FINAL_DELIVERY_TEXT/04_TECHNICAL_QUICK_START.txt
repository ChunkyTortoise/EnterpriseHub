FASTEST START
--------------------------------------------------------------------------------
1. Run the automated setup script:
   ./setup_demo.sh

2. Follow the prompts, then start the dashboard:
   source venv/bin/activate
   cd ghl_real_estate_ai/streamlit_demo
   streamlit run app.py

Open browser to: http://localhost:8501

API KEYS SETUP (For Full Functionality)
--------------------------------------------------------------------------------
1. Claude AI (Anthropic)
   - Get key from: https://console.anthropic.com/
   - Add to .env file: ANTHROPIC_API_KEY=your_key_here

2. GoHighLevel CRM
   - Get from: GHL Settings -> API
   - Add to .env file: GHL_API_KEY=your_key_here
   - Add to .env file: GHL_LOCATION_ID=your_id_here

WHAT WORKS WITHOUT API KEYS
--------------------------------------------------------------------------------
- Full UI demonstration
- Demo data exploration ($2.4M pipeline)
- All component interactions
- Performance analytics
- Document templates
- Workflow visualizations
