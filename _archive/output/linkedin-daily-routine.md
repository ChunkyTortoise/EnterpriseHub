# LinkedIn Daily Job Hunt Routine — 15 min/day

> **Goal**: Land first paying gig ASAP via LinkedIn + direct applications
> **Started**: Feb 26, 2026

---

## Daily 15-Min Block (do in order)

### 1. Search (3 min) — run all 5 strings, filter "Past 24 hours" + Remote
```
"AI engineer" OR "LLM engineer" OR "RAG engineer" remote Python
"ML engineer" OR "machine learning engineer" remote contract
"AI automation" OR "AI integration" remote Python
GoHighLevel OR "CRM automation" OR "AI chatbot" freelance
"applied AI" OR "AI platform" remote senior
```
**Filters**: Remote | Past Week | United States | Contract OR Full-time

### 2. Apply (5 min) — Easy Apply to 3-5 matching roles
Add a custom note using Template A or B (see below).

### 3. Connect (4 min) — 5 targeted connection requests to hiring managers/CTOs
Use Template A for AI companies, B for startups, C for GHL/CRM.

### 4. Engage (2 min) — Comment on 3-5 AI/tech leader posts
Add real value — not "great post!". Share a data point, ask a question, or add a counterpoint.

### 5. Post (1 min) — Publish from week 2-5 content calendar
Files: `content/linkedin/week2-5/`

## Connection Request Templates
