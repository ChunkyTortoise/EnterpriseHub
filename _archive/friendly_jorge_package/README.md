## üöÄ Quick Start

### 1. Prerequisites
- Python 3.9+
- PostgreSQL database
- GHL (GoHighLevel) account
- California DRE license
- Rancho Cucamonga market knowledge

### 2. Installation

```bash
# Clone or copy the friendly_jorge_package
cd friendly_jorge_package

# Install dependencies
pip install -r requirements.txt

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# Run automated setup
python scripts/setup_friendly_bot.py

# Test the friendly approach
python scripts/test_friendly_approach.py
```

### 3. Configuration

```python
# Update jorge_friendly_config.py with your settings
RANCHO_CUCAMONGA_FOCUS = True
FRIENDLY_APPROACH = True
CA_DRE_LICENSE = "your_dre_license_number"
BROKERAGE_NAME = "Your Brokerage Name"
```

### 4. Launch Dashboard

```bash
streamlit run jorge_friendly_dashboard.py --server.port 8501
```
