## ðŸ“‹ Screenshot Guidelines

### Technical Specs
- **Resolution**: 1920x1080 or 2560x1440 (for retina displays)
- **Format**: PNG (lossless, best for UI)
- **File naming**: `[module]_[feature]_[number].png`
  - Example: `margin_hunter_sensitivity_heatmap_01.png`
- **File size**: Optimize with tools like TinyPNG (aim for <500KB each)

### Content Guidelines
1. **Use realistic data**: Not "test" or "lorem ipsum"
2. **Professional appearance**: Clean UI, no errors/warnings
3. **Highlight key features**: Show the module's value
4. **Consistent branding**: Same theme/settings across all screenshots
5. **No sensitive info**: No real API keys, personal data

### Capture Process
1. Run the application: `streamlit run app.py`
2. Navigate to each module
3. Set up realistic scenario/data
4. Use screenshot tool (macOS: Cmd+Shift+4, Windows: Snipping Tool)
5. Crop to remove unnecessary chrome
6. Save to this directory with descriptive name
7. Optimize file size
8. Update checklist above
