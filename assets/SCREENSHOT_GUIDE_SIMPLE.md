## If Something Goes Wrong

**Module won't load:**
- Refresh the page (F5)
- Wait 5 seconds
- Try again

**Inputs are blank:**
- Type sample values: ticker "SPY", price "$99", etc.
- Continue with screenshot

**Charts aren't showing:**
- Wait 3-5 seconds (Plotly takes time to render)
- Refresh and try again
- Scroll down to see if chart is below fold

**Screenshot too dark/blurry:**
- Make sure browser zoom is at 100% (Ctrl+0)
- Try again with better lighting
- Or adjust screenshot tool brightness
