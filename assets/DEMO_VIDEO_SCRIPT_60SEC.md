## RECORDING TIPS

### Audio Tips
1. **Speak 20% slower** than normal conversation
2. **Pause 0.5 seconds** between sentences (easier to edit)
3. **Emphasize these phrases** (speak slightly louder):
   - "$24,000 at Bloomberg"
   - "100 pricing scenarios"
   - "2 minutes instead of 2 hours"
   - "220 automated tests"
   - "$0.003 per post"

4. **Vocal Energy:**
   - Start HIGH (hook)
   - Stay MEDIUM (demos)
   - End HIGH (CTA)

5. **Avoid filler words:** No "um", "uh", "like", "you know"
   - If you mess up, pause 2 seconds and restart the sentence
   - Easy to edit out in post-production

### Visual Tips
1. **Mouse cursor:** Move slowly (jerky cursor looks bad)
2. **Transitions:** Let page load 0.5 seconds before continuing
3. **Loading screens:** Keep spinners in video (proves it's real)
4. **Text readability:** All text must be readable at 1080p

## TROUBLESHOOTING

**Problem: Audio too quiet**
- Adjust microphone position (6-8 inches from mouth)
- Or use phone earbuds as mic (better quality)
- Or record again with normalized audio in editing

**Problem: Speaking too fast**
- Record again, slower pace
- Or use editing software to slow down audio to 0.9x speed

**Problem: Module takes too long to load**
- Record late at night (less Streamlit Cloud traffic)
- Or pre-load all modules in browser tabs
- Or speed up loading screen in editing (2x)

**Problem: Background noise visible**
- Use editing software to remove noise
- Or record again with windows closed

**Problem: File too large**
- Use Handbrake (free compression tool)
- Or reduce bitrate to 5 Mbps in export settings
