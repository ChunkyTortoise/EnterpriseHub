# Demo Video Shooting Guide - 60-Second Version

## Pre-Production Checklist (15 minutes)

### Equipment Setup
- [ ] **Screen recording software installed:**
  - **Mac:** QuickTime (built-in), Kap (free), or OBS Studio (advanced)
  - **Windows:** Windows Game Bar (Win+G), OBS Studio, or Camtasia
  - **Cross-platform:** Loom (easiest, cloud-based)

- [ ] **Microphone test:**
  - Record 10-second test: "Testing, one, two, three"
  - Play back: Is it clear? No background noise?
  - Position mic 6-8 inches from mouth
  - Close windows (reduce ambient noise)

- [ ] **Browser setup:**
  - **Resolution:** 1920x1080 (Full HD)
  - **Zoom:** 100% (CMD+0 / CTRL+0)
  - **Mode:** Incognito/private (clean browser, no extensions)
  - **Pre-load demo:** https://enterprise-app-mwrxqf7cccewnomrbhjttf.streamlit.app/

- [ ] **Recording settings:**
  - **Frame rate:** 30 FPS minimum
  - **Quality:** 1080p (not 4K - file size too large)
  - **Audio:** 44.1kHz, mono or stereo

### Script Preparation
- [ ] Print script (don't read from screen while recording)
- [ ] Practice 2-3 times (aim for natural delivery)
- [ ] Highlight KEY WORDS for emphasis
- [ ] Time yourself (should be 55-65 seconds with pauses)

## Recording Best Practices

### Audio Tips
1. **Speak 20% slower** than normal conversation (clarity over speed)
2. **Pause 0.5 seconds** between sentences (easier to edit)
3. **Emphasize KEY WORDS:**
   - "$24,000 at Bloomberg"
   - "100 pricing scenarios"
   - "2 minutes instead of 2 hours"
   - "220 automated tests"
   - "$0.003 per post"

4. **Vocal Energy:**
   - Start HIGH energy (hook)
   - Maintain MEDIUM energy (demos)
   - End HIGH energy (CTA)

5. **Avoid Filler Words:**
   - Don't say "um", "uh", "like", "you know"
   - If you mess up, pause 2 seconds, restart sentence (easy to edit)

### Visual Tips
1. **Cursor Movement:**
   - Move cursor SLOWLY (jerky cursor looks unprofessional)
   - Hide cursor when showing visuals (press F11 for fullscreen)

2. **Transitions:**
   - Sidebar clicks should be VISIBLE (don't cut too fast)
   - Wait 0.5 seconds after page loads before continuing

3. **Loading Screens:**
   - Keep loading spinners IN the video (shows it's real)
   - But no longer than 2 seconds (speed up in editing if needed)

4. **Text Readability:**
   - All text must be readable at 1080p
   - If text too small, zoom in (150-200%)

### Step 1: Do a Full Rehearsal (10 minutes)

1. Start screen recording software

**Expected:** You'll need 2-4 takes. That's normal.

- Take 1: Audio too quiet (adjust mic)

## Post-Production (Light Editing - 15 minutes)

### Editing Checklist

**Tool Options:**
- **Easy:** Loom (built-in editor), iMovie (Mac), Windows Video Editor (PC)
- **Pro:** DaVinci Resolve (free), Adobe Premiere Pro

### Edits to Make:

1. **Trim Intro/Outro:**
   - Cut 2-3 seconds before you start speaking
   - Cut immediately after "Full source code on GitHub"

2. **Add Captions (Highly Recommended):**
   - 80% of LinkedIn videos watched with sound OFF
   - Use: Kapwing (auto-captions, free), Rev.com ($1/min), or manual
   - Font: Sans-serif, white text with black outline
   - Size: Large (1/4 screen height)

3. **Add Text Overlays:**
   - At 0:02: "$24,000 at Bloomberg" (emphasize hook)
   - At 0:10: "100 scenarios" (over heatmap)
   - At 0:30: "2 min vs 2 hours" (over Data Detective)
   - At 0:40: "$0.003 per post" (over Content Engine)
   - At 0:50: "220+ tests, 85% coverage" (over GitHub)
   - At 0:57: "github.com/ChunkyTortoise/enterprise-hub"

4. **Speed Adjustments:**
   - If loading screen >2 seconds: Speed up to 2x
   - If total video >65 seconds: Speed up slow sections to 1.2x

5. **Audio Enhancements:**
   - Normalize audio levels (all speech same volume)
   - Remove background noise (if any)
   - Compress audio (reduce dynamic range for consistent listening)

6. **Add Background Music (Optional):**
   - Use: YouTube Audio Library, Epidemic Sound, or Artlist
   - Volume: -20dB (very quiet, should not distract from narration)
   - Style: Upbeat, tech-focused (not dramatic)

### Export Settings
- **Format:** MP4 (H.264 codec)
- **Resolution:** 1920x1080 (1080p)
- **Frame Rate:** 30 FPS
- **Bitrate:** 5-8 Mbps (good quality, reasonable file size)
- **File Size Target:** Under 50 MB (LinkedIn limit is 5 GB, but smaller uploads faster)

## Troubleshooting

### Issue: Audio quality poor
**Solution:**
- Use headphone mic (built-in laptop mics are bad)
- Or use phone as external mic (voice memo app ‚Üí AirDrop)
- Or buy: Blue Yeti ($100) or Rode NT-USB ($169)

### Issue: Screen recording laggy
**Solution:**
- Close all other apps (especially Chrome with many tabs)
- Record at 720p instead of 1080p, upscale in editing
- Use OBS Studio (more efficient than QuickTime)

### Issue: File size too large
**Solution:**
- Use Handbrake (free) to compress: Target 5-8 Mbps bitrate
- Or use online tool: CloudConvert, FreeConvert

### Issue: Modules not loading fast
**Solution:**
- Record late at night / early morning (less Streamlit Cloud traffic)
- Or pre-load all modules in browser tabs before recording
- Speed up loading screens in editing (2x speed)

### Issue: Voice sounds nervous
**Solution:**
- Practice script 5+ times (muscle memory)
- Smile while talking (sounds more confident)
- Stand up while recording (better breath control)
- If still nervous: Record video first, add voiceover after

## Pro Tips

1. **Record in Batches:**
   - Record Scenes 1-3 in one take
   - Record Scenes 4-7 in another take
   - Splice together (easier than perfect 60-second take)

2. **Voiceover vs Live:**
   - **Live:** Record voice + screen simultaneously (feels more authentic)
   - **Voiceover:** Record screen first, add voice after (more control)
   - **Recommendation:** Try live first (faster), voiceover if issues

3. **B-Roll Footage:**
   - Record 2-3 minutes of raw footage (navigating modules)
   - Cut down to 60 seconds in editing
   - Easier than nailing perfect timing live

4. **Thumbnail Matters:**
   - YouTube/LinkedIn use first frame as thumbnail
   - Make first frame VISUALLY INTERESTING (not blank screen)
   - Or upload custom thumbnail: Use `margin-hunter-heatmap.png`

5. **Engagement Boost:**
   - Post video at 8-10 AM EST on Tuesday-Thursday
   - Reply to every comment in first hour (algorithm boost)
